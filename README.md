### Installation

##### Clone repo
`git clone git@github.com:albertorsesc/app.houseify.io.git`

##### Install Dependencies
`composer install && npm install`

##### Create Environment file
`cp ./.env.example .env`

##### Setup DB variables

##### Seed database
`php artisan migrate --seed`

##### Start app
* `npm run watch`
* `php artisan serve`

##### Visit app URL
`localhost:3000`
